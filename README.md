# NIDCAP és N-PASS megfigyelések elemzései

BSc szakdolgozathoz készített elemző kód, amely a kenguruzás hatását vizsgálja koraszülöttek komfortérzetére NIDCAP megfigyelések és N-PASS fájdalomskála alapján.

## Elemzések

1. Stressz- és önszabályozó jelek változása (Wilcoxon-próba)
2. NIDCAP–N-PASS korreláció (Spearman)
3. NIDCAP alrendszerek összehasonlítása
4. Alvás-ébrenlét állapotok változása
